# Install .NET Framework 4.7.2 Developer Pack

You need to install the .NET Framework 4.7.2 Developer Pack to build this project.

## Quick Install:

**Direct download link:**
https://dotnet.microsoft.com/download/dotnet-framework/net472

Scroll down to "Developer Pack" and download it. Run the installer.

**OR**

1. Go to: https://aka.ms/msbuild/developerpacks
2. Download ".NET Framework 4.7.2 Developer Pack"
3. Install it
4. Rebuild the project

After installation, try building again!
